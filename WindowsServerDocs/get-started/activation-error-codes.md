# <a name="troubleshooting-activation-error-codes"></a>Solución de problemas de códigos de error de activación

**Usuarios domésticos**. Este artículo está pensado para su uso por parte de los agentes de soporte técnico y profesionales de TI. Para más información acerca de los mensajes de error de activación de Windows, vaya a los siguientes sitios web de Microsoft:

[Obtener ayuda con los errores de activación de Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Si intenta usar una clave de activación múltiple (MAK) o el Servicio de administración de claves (KMS) para realizar la activación por volumen en uno o varios equipos basados en Windows, recibirá mensajes de error que contienen códigos de error específicos. En este artículo se describe cómo solucionar estos errores.
