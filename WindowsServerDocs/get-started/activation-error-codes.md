# <a name="troubleshooting-activation-error-codes"></a>Solución de problemas de códigos de Error de activación

**Usuarios domésticos** este artículo está pensado para su uso por los agentes de soporte técnico y los profesionales de TI. Si desea obtener más información acerca de los mensajes de error de activación de Windows, vaya a los siguientes sitios Web de Microsoft:

[Obtener ayuda con los errores de activación de Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Al intentar usar una clave de activación múltiple (MAK) o el servicio de administración de claves (KMS) para realizar la activación por volumen en uno o más equipos basados en Windows, recibe mensajes de error que contienen códigos de error específicos. En este artículo se describe cómo solucionar estos errores.
