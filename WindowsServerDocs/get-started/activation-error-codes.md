# Solución de problemas de códigos de Error de activación

**Usuarios de inicio** Este artículo está pensado para su uso por los agentes de soporte técnico y los profesionales de TI. Si buscas obtener más información sobre los mensajes de error de activación de Windows, ve a los sitios Web de Microsoft siguientes:

[Obtener ayuda sobre los errores de activación de Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Si intentas usar una clave de activación múltiple (MAK) o el servicio de administración de claves (KMS) para realizar la activación por volumen en uno o más equipos basados en Windows, recibirás los mensajes de error que contienen los códigos de error específicos. En este artículo se describe cómo solucionar estos errores.
