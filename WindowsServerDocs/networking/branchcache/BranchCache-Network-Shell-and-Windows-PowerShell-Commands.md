# <a name="branchcache-network-shell-and-windows-powershell-commands"></a>El Shell de red BranchCache y comandos de Windows PowerShell

>Se aplica a: Windows Server (punto y anual canal), Windows Server 2016

En Windows Server 2016, puedes configurar y administrar BranchCache mediante Windows PowerShell o los comandos de Shell de red (Netsh) para BranchCache.  
  
En futuras versiones de Windows, Microsoft puede quitar la funcionalidad de netsh para BranchCache. Microsoft recomienda que crean transiciones Windows PowerShell si actualmente usas netsh para configurar y administrar BranchCache y otras tecnologías de redes.  
  
Referencias de comandos de Windows PowerShell y netsh están en las siguientes ubicaciones. Aunque ambas referencias de comando publicadas para sistemas operativos anteriores a Windows Server 2016, estas referencias son precisas para este sistema operativo.  
  
-   [Comandos Netsh para BranchCache en Windows Server 2008 R2](https://technet.microsoft.com/library/dd979561(v=ws.10))  
  
-   [BranchCache Cmdlets de Windows PowerShell](https://technet.microsoft.com/library/hh848392.aspx) para Windows Server 2012.  
  
> [!TIP]  
> Para ver una lista de comandos de Windows PowerShell para BranchCache en el símbolo del sistema de Windows PowerShell, escribe `Get-Command -Module BranchCache` en el símbolo del sistema de Windows PowerShell y, a continuación, presiona ENTRAR.
