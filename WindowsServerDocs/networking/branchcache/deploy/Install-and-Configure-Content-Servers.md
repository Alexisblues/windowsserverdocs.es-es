# <a name="install-and-configure-content-servers"></a>Instalar y configurar servidores de contenido

>Se aplica a: Windows Server (canal semianual), Windows Server 2016

Cuando se implementa BranchCache en modo caché distribuida o modo caché hospedada, debe implementar uno o más servidores de contenido en la oficina central o en la nube. Los servidores de contenido que son servidores web o servidores de aplicaciones utilizan la característica BranchCache. Servidores de contenido que son servidores de archivos utilizan BranchCache para el servicio de rol de los archivos de red del rol de servidor Servicios de archivos en Windows Server 2016.  
  
Vea los temas siguientes para implementar servidores de contenido.  
  
-   [Instalar a servidores de contenido que utilizan la característica BranchCache](../../branchcache/deploy/Install-Content-Servers-that-Use-the-BranchCache-Feature.md)  
  
-   [Instalar servidores de contenido de servicios de archivo](../../branchcache/deploy/Install-File-Services-Content-Servers.md)
