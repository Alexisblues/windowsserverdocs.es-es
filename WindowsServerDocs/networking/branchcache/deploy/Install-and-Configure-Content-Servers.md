# <a name="install-and-configure-content-servers"></a>Instalar y configurar los servidores de contenido

>Se aplica a: Windows Server (punto y anual canal), Windows Server 2016

Cuando implementas BranchCache en modo de caché distribuida o memoria caché hospedada, debes implementar uno o varios servidores de contenido en la oficina principal o en la nube. Los servidores de contenido que son servidores Web o servidores de la aplicación usan la característica BranchCache. Los servidores de contenido que son servidores de archivos usan el BranchCache para el servicio de rol de archivos de red del rol de servidor de servicios de archivos en Windows Server 2016.  
  
Consulta los temas siguientes para implementar los servidores de contenido.  
  
-   [Instalar a contenido los servidores que usan la característica BranchCache](../../branchcache/deploy/Install-Content-Servers-that-Use-the-BranchCache-Feature.md)  
  
-   [Instalar los servidores de contenido de los servicios de archivo](../../branchcache/deploy/Install-File-Services-Content-Servers.md)
