# <a name="enable-hash-publication-for-file-servers"></a>Habilitar el Hash de publicación para servidores de archivos

>Se aplica a: Windows Server (punto y anual canal), Windows Server 2016

Puedes habilitar BranchCache publicación de hash en un servidor de archivos o en varios servidores de archivos.  
  
-   Para habilitar la publicación de hash en un servidor de archivos mediante la directiva de grupo del equipo local, consulta [habilitar Hash de publicación para servidores de archivos de miembro de dominio no](../../branchcache/deploy/Enable-Hash-Publication-for-Non-Domain-Member-File-Servers.md).  
  
-   Para habilitar la publicación de hash en varios servidores de archivos con la directiva de grupo de dominio, consulta [habilitar Hash de publicación para servidores de archivos de miembro de dominio](../../branchcache/deploy/Enable-Hash-Publication-for-Domain-Member-File-Servers.md).  
  
> [!NOTE]  
> Si tienes varios servidores de archivos y quieres habilitar la publicación de hash por compartir, en lugar de publicación de hash habilitar para todos los recursos compartidos, puedes usar las instrucciones en el tema [habilitar Hash de publicación para servidores de archivos de miembro de dominio no](Enable-Hash-Publication-for-Non-Domain-Member-File-Servers.md).
