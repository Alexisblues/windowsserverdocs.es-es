# <a name="install-content-servers-that-use-the-branchcache-feature"></a>Instalar a contenido los servidores que usan la característica BranchCache

>Se aplica a: Windows Server (punto y anual canal), Windows Server 2016

Para implementar los servidores de contenido que son servidores Web de protocolo seguro de transferencia de hipertexto (HTTPS), los servidores Web de protocolo de transferencia de hipertexto (HTTP) y servicio de transferencia inteligente en segundo plano (BITS)-servidores del sistema de sitios de la aplicación basada en los servidores, como la distribución de rama de Windows Server Update Services (WSUS) y System Center Configuration Manager, debes instalar la característica BranchCache, iniciar el servicio BranchCache y (para servidores WSUS), realiza los pasos de configuración adicional.  
  
Consulta los temas siguientes para implementar los servidores de contenido.  
  
-   [Instalar la característica BranchCache](Install-the-BranchCache-Feature.md)  
  
-   [Configurar Windows Server Update Services & #40; WSUS & #41; Servidores de contenido](configure-wsus-content-servers.md)
