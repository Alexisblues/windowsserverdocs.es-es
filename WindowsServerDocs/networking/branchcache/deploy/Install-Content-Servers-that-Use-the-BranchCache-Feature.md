# <a name="install-content-servers-that-use-the-branchcache-feature"></a>Instalar servidores de contenido que utilizan la característica BranchCache

>Se aplica a: Windows Server (canal semianual), Windows Server 2016

Para implementar servidores de contenido que son servidores Web de protocolo de transferencia de hipertexto seguro (HTTPS), los servidores Web de protocolo de transferencia de hipertexto (HTTP) y servicio de transferencia inteligente en segundo plano (BITS): en función de los servidores de aplicaciones, como Windows Server Update Services () WSUS) y servidores de sistema de sitio de distribución de la rama de System Center Configuration Manager, debe instalar la característica BranchCache, iniciar el servicio BranchCache y (para servidores WSUS), realice los pasos de configuración adicionales.  
  
Vea los temas siguientes para implementar servidores de contenido.  
  
-   [Instalar la característica BranchCache](Install-the-BranchCache-Feature.md)  
  
-   [Configurar Windows Server Update Services &#40;WSUS&#41; servidores de contenido](configure-wsus-content-servers.md)
