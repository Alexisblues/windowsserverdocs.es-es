# <a name="ip-address-management-ipam"></a>Administración de direcciones IP (IPAM)

>Se aplica a: Windows Server (canal semianual), Windows Server 2016

Administración de direcciones IP (IPAM) es un conjunto integrado de herramientas para habilitar-to-end planeación, implementación, administración y supervisión de la infraestructura de direcciones IP, con una experiencia de usuario enriquecida. IPAM detecta automáticamente los servidores con infraestructura de direcciones IP y los servidores del Sistema de nombres de dominio (DNS) de la red y permite administrarlos desde una interfaz central.
