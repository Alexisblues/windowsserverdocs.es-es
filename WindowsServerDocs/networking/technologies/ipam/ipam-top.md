# <a name="ip-address-management-ipam"></a>Administración de direcciones IP (IPAM)

>Se aplica a: Windows Server (punto y anual canal), Windows Server 2016

Administración de direcciones IP (IPAM) es un conjunto integrado de herramientas para habilitar-to-end planificación, implementación, administración y supervisión de la infraestructura de dirección IP, con una experiencia de usuario enriquecida. IPAM automáticamente descubre los servidores de infraestructura de direcciones IP y los servidores de sistema de nombres de dominio (DNS) en la red y te permite administrar a ellos desde una interfaz central.
