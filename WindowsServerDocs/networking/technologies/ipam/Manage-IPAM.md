# <a name="manage-ipam"></a>Administrar IPAM

>Se aplica a: Windows Server (canal semianual), Windows Server 2016

Esta guía proporciona información para solucionar problemas de la característica de administración de direcciones IP (IPAM) en Windows Server 2016 y de administración.  
  
En Windows Server 2016, IPAM admite el registro de recursos DNS, el reenviador condicional y la administración de zonas DNS para los servidores DNS unido al dominio, integrado en Active Directory y basado en archivos. Además, IPAM admite el control de acceso basado en roles y toda la funcionalidad en versiones anteriores de la tecnología.  
  
Esta guía incluye las siguientes secciones:  
  
-   [Administración de registros de recursos DNS](../../technologies/ipam/DNS-Resource-Record-Management.md)  
  
-   [Administración de zonas DNS](../../technologies/ipam/DNS-Zone-Management.md)  
  
-   [Administrar recursos en varios bosques de Active Directory](../../technologies/ipam/Manage-Resources-in-Multiple-Active-Directory-Forests.md)  
  
-  [Purgar datos de uso](../../technologies/ipam/Purge-Utilization-Data.md)  
  
-   [Control de acceso basado en roles](../../technologies/ipam/Role-based-Access-Control.md)
