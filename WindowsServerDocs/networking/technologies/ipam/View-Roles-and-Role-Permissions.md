# <a name="view-roles-and-role-permissions"></a>Observa los Roles y permisos de rol

>Se aplica a: Windows Server (punto y anual canal), Windows Server 2016

Puedes usar este tema para ver roles de usuario de Control de acceso en la consola IPAM del cliente.  
  
Pertenencia a **administradores**, o equivalente, es lo mínimo necesario para realizar este procedimiento.  
  
### <a name="to-view-access-control-roles"></a>Para ver las funciones de Control de acceso  
  
1.  En el administrador del servidor, haz clic en **IPAM**. Aparece la consola IPAM del cliente.  
  
2.  En el panel de navegación, haz clic en **el CONTROL de acceso**.  
  
3.  En el panel de navegación inferior, haga clic en **Roles**. En el panel, se enumeran los roles.  
  
    ![En el panel se enumeran los roles](../../media/View-Roles-and-Role-Permissions/ipam_ViewRoles_01.jpg)  
  
4.  Selecciona la función cuyos permisos que quieras ver. En el panel de detalles inferior, se muestran las operaciones que tienen permiso para la función.  
  
    ![Permisos de la función de vista](../../media/View-Roles-and-Role-Permissions/ipam_ViewRoles_02.jpg)
