# <a name="view-dns-resource-records-for-a-specific-ip-address"></a>Ver registros de recursos DNS para obtener una dirección IP específica

>Se aplica a: Windows Server (punto y anual canal), Windows Server 2016

Puedes usar este tema para ver los registros de recursos DNS que se asocian con la dirección IP que elijas.  
  
Pertenencia a **administradores**, o equivalente, es lo mínimo necesario para realizar este procedimiento.  
  
### <a name="to-view-resource-records-for-an-ip-address"></a>Para ver los registros de recursos de una dirección IP  
  
1.  En el administrador del servidor, haz clic en **IPAM**. Aparece la consola IPAM del cliente.  
  
2.  En el panel de navegación, en **espacio de direcciones IP**, haz clic en **inventario de dirección IP**. En el panel de navegación inferior, haga clic en **IPv4** o **IPv6**. El inventario de la dirección IP aparece en la vista de búsqueda del panel de visualización. Busca y selecciona la dirección IP cuyos registros de recursos DNS que deseas ver.  
  
    ![Inventario de dirección IP de vista](../../media/View-DNS-Resource-Records-for-a-Specific-IP-Address/ipam_IPInventory_01.jpg)  
  
3.  En el panel **vista detalles**, haz clic en **registros de recursos DNS**. Se muestran los registros de recursos que están asociados a la dirección IP seleccionada.  
  
    ![Ver registros de recursos DNS](../../media/View-DNS-Resource-Records-for-a-Specific-IP-Address/ipam_IPInventory_02.jpg)
