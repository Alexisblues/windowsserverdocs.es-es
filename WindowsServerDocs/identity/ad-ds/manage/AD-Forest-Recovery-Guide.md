# <a name="active-directory-forest-recovery-guide"></a>Guía de recuperación de bosques de Active Directory

>Se aplica a: Windows Server 2016, Windows Server 2012 y 2012 R2, Windows Server 2008 y 2008 R2, Windows Server 2003

Esta guía contiene prácticas recomendadas para la recuperación de un bosque Active Directory® si el error en todo el bosque representa todos los controladores de dominio (DC) en el bosque no sea capaz de funcionar con normalidad. Los pasos que contiene servir como plantilla para el plan de recuperación del bosque, que puede personalizar para su entorno particular. Estos pasos se aplican a los controladores de dominio que ejecutan Microsoft® Windows Server 2016, 2012 R2, 2012, 2008 R2, 2008 y 2003 sistemas operativos.  
  
> [!NOTE]
> Los procedimientos que son únicos para los controladores de dominio que ejecutan Windows Server 2003 se consolidan en [recuperación de bosque de AD Windows Server 2003](AD-Forest-Recovery-Windows-Server-2003.md).
