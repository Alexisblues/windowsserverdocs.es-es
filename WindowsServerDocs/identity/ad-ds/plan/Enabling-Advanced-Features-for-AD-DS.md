# <a name="enabling-advanced-features-for-ad-ds"></a>Habilitar las características avanzadas de AD DS

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Los servicios de dominio de Active Directory (AD DS) permite introducir características avanzadas en el entorno generando los niveles funcionales de dominio o bosque. Para usar las características avanzadas de AD DS, debe identificar los sistemas operativos que se ejecutan en los controladores de dominio en su entorno.   
  
Debes determinar también el mejor nivel funcional de la organización basándose en la infraestructura existente y, a continuación, aumentar el nivel funcional de dominio o bosque según corresponda. Puedes generar el nivel funcional cuando todos los controladores de dominio en el dominio o bosque ejecutan una versión adecuada de Windows. Aunque aumentar el nivel funcional hace posible para poder habilitar nuevas características, también limita las versiones de sistemas operativos Windows que se pueden ejecutar en controladores de dominio en su entorno.
