# <a name="enabling-advanced-features-for-ad-ds"></a>Habilitación de características avanzadas para AD DS

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Los servicios de dominio de Active Directory (AD DS) permite introducir características avanzadas en su entorno al elevar los niveles funcionales de dominio o bosque. Para usar características avanzadas de AD DS, debe identificar los sistemas operativos que se ejecutan en los controladores de dominio en su entorno.   
  
Debe determinar también el nivel funcional del mejor para su organización en función de la infraestructura existente y, a continuación, elevar el nivel funcional de dominio o bosque según corresponda. Puede elevar el nivel funcional cuando todos los controladores de dominio en el dominio o bosque ejecutan una versión adecuada de Windows. Aunque elevar el nivel funcional permite habilitar las nuevas características, también limita las versiones de sistemas operativos de Windows que se pueden ejecutar en controladores de dominio en su entorno.
