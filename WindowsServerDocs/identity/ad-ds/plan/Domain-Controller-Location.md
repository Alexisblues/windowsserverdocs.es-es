# <a name="domain-controller-location"></a>Ubicación del controlador de dominio

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Los clientes usan sistema de nombres de dominio (DNS) para buscar controladores de dominio para operaciones completas, como el procesamiento de solicitudes de inicio de sesión o buscar en el directorio de recursos publicados. Controladores de dominio registran una variedad de registros de DNS para ayudar a los clientes y otros equipos localizan. Estos registros se denominan colectivamente los registros de localizador.  
  
Controladores de dominio, también usan DNS para buscar otros controladores de dominio y para realizar tareas como la replicación. El proceso por el que los controladores de dominio buscar otros controladores de dominio es el mismo que el proceso por el que los clientes localizan controladores de dominio.
