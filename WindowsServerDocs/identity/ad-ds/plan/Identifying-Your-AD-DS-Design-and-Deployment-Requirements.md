# <a name="identifying-your-ad-ds-design-and-deployment-requirements"></a>Identificar el diseño de AD DS y los requisitos de implementación

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Realizar una evaluación de alto nivel de su entorno actual e identificar correctamente las tareas de implementación de servicios de dominio de Active Directory (AD DS) son fundamental para el éxito de tu estrategia de implementación de AD DS.  
  
La estrategia de implementación de AD DS depende de la configuración de red existente. Por ejemplo, si tu organización actualmente ejecuta Windows Server 2003, puede actualizar el sistema operativo a Windows Server 2008. El proceso de implementación puede implicar reestructuración de dominios, dentro de un bosque de Active Directory o entre bosques de Active Directory existentes. Puede que tengas que reestructurar los dominios existentes después de implementar Windows Server 2008 AD DS o después de cambios en la organización o adquisiciones corporativas.

-   [Requisitos de implementación de AD DS](../../ad-ds/plan/AD-DS-Deployment-Requirements.md)
