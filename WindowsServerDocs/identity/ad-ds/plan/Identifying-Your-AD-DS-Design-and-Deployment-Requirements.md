# <a name="identifying-your-ad-ds-design-and-deployment-requirements"></a>Determinación del diseño de AD DS y requisitos de implementación

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Realizar una evaluación detallada del entorno actual e identificar correctamente las tareas de implementación de servicios de dominio de Active Directory (AD DS) son esencial para el éxito de su estrategia de implementación de AD DS.  
  
La estrategia de implementación de AD DS depende de la configuración de red existente. Por ejemplo, si su organización actualmente ejecuta Windows Server 2003, puede actualizar el sistema operativo a Windows Server 2008. El proceso de implementación puede implicar la reestructuración de dominios existentes, dentro de un bosque de Active Directory o entre bosques de Active Directory. Es posible que deba reestructurar los dominios existentes después de implementar AD DS de Windows Server 2008 o después de los cambios organizativos o adquisiciones corporativas.
