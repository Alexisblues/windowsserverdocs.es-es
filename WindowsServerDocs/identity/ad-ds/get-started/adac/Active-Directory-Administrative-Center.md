# <a name="active-directory-administrative-center"></a>Centro de administración de Active Directory

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

El Active Directory centro de administración (ADAC) en Windows Server incluye características de experiencia de administración mejoradas. Estas características facilitan las tareas para administrar los servicios de dominio de Active Directory (AD DS).
