# <a name="active-directory-administrative-center"></a>Centro de administración de Active Directory

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

El Centro de administración de Active Directory (ADAC) de Windows Server incluye características mejoradas para la experiencia de administración. Estas características facilitan la carga administrativa de administrar Active Directory Domain Services (AD DS).
