# <a name="dynamic-access-control-scenario-overview"></a>Control de acceso dinámico: Información general del escenario

>Se aplica a: Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

En Windows Server 2012, puedes aplicar el control de datos a través de los servidores de archivos para controlar quién puede acceder a información y para auditar quién tiene acceso a la información. Control de acceso dinámico te permite:  
  
-   Identificar los datos mediante el uso de clasificación manual y automática de archivos. Por ejemplo, podría etiquetar datos en los servidores de archivos en toda la organización.  
  
-   Controlar el acceso a archivos mediante la aplicación de directivas de seguridad de red que usan directivas de acceso central. Por ejemplo, podrías definir quién puede acceder a información de estado dentro de la organización.  
  
-   Auditar el acceso a archivos mediante las directivas de auditoría central de cumplimiento informes y el análisis detallado. Por ejemplo, puede identificar quién tiene acceso a información confidencial.  
  
-   Aplicar la protección de Rights Management Services (RMS) mediante el uso de cifrado automático de RMS de documentos con información confidencial de Microsoft Office. Por ejemplo, podría configurar RMS para cifrar todos los documentos que contienen información portabilidad seguros de salud y responsabilidad (HIPAA).  
  
El conjunto de características de Control de acceso dinámico se basa en las inversiones de infraestructura que pueden ser aplicaciones más usadas por partners y line-of-business y las características pueden proporcionar gran valor para las organizaciones que usan Active Directory. Esta infraestructura incluye:  
  
-   Un nuevo motor autorización y auditoría de Windows, que puede procesar expresiones condicionales y directivas central.  
  
-   Compatibilidad con la autenticación de Kerberos para notificaciones de usuario y dispositivo.  
  
-   Mejoras en la infraestructura de clasificación de archivo (FCI).  
  
-   Extensibilidad de RMS soporte técnico para que partners puedan proporcionar soluciones que cifrar archivos no son de Microsoft.
