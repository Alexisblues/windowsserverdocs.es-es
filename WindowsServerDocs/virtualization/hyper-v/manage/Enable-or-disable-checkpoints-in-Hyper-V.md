# <a name="enable-or-disable-checkpoints-in-hyper-v"></a>Habilitar o deshabilitar puntos de control en Hyper-V

>Se aplica a: Windows 10, Windows Server 2016 y Microsoft Hyper-V Server 2016, Windows Server 2019, Microsoft Hyper-V Server 2019
  
Puede elegir habilitar o deshabilitar los puntos de control para cada máquina virtual.  
  
1.  En **Administrador de Hyper-V**, haga clic en la máquina virtual y haga clic en **configuración**.  
  
2.  En el **administración** sección, seleccione **puntos de control**.  
  
3.  Para permitir que los puntos de control que se debe realizar de esta máquina virtual, asegúrese de que **Habilitar puntos de comprobación** está seleccionada. Para deshabilitar los puntos de control, desactive la **Habilitar puntos de comprobación** casilla de verificación.  
  
4.  Haga clic en **Aplicar** para aplicar los cambios. Si ya ha terminado, haga clic en **Aceptar** para cerrar el cuadro de diálogo.
