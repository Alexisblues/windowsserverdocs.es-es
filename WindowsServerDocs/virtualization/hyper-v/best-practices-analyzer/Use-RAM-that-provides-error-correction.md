# <a name="use-ram-that-provides-error-correction"></a>Uso de RAM que proporciona la corrección de errores

>Se aplica a: Windows Server 2016

Para más información acerca de los análisis y los procedimientos recomendados, vea [Analizador de procedimientos recomendados](https://go.microsoft.com/fwlink/?LinkId=122786).  
  
|Property|Detalles|  
|-|-|  
|**Sistema operativo**|Windows Server 2016|  
|**Característica del producto**|Hyper-V|  
|**Gravedad**|Error|  
|**Categoría**|Configuración|  
  
En las secciones siguientes, la cursiva indica texto de la interfaz de usuario que aparece en la herramienta Best Practices Analyzer para resolver este problema.  
  
## <a name="issue"></a>Problema  
  
*La RAM en uso en este equipo es no corrección de errores (ECC) de RAM.*  
  
## <a name="impact"></a>Impacto  
  
*Microsoft no admite Windows Server 2016 en equipos sin RAM de corrección de errores.*  
  
## <a name="resolution"></a>Resolución  
  
*Compruebe el servidor se incluyen en el catálogo de Windows Server y calificado para Hyper-V.*  
  
Para comprobar si aparece el servidor, consulte el [catálogo de Windows Server](https://www.windowsservercatalog.com/).
