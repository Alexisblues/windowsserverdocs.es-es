# <a name="best-practices-analyzer-for-hyper-v"></a>Analizador de procedimientos recomendados para Hyper-V

>Se aplica a: Windows Server 2016
  
En la administración de Windows, los *procedimientos recomendados* son instrucciones que se consideran ideales, en circunstancias normales, para configurar un servidor, según la definición de los expertos. Seguir los procedimientos recomendados, incluso los fundamentales, no necesariamente pueden causar problemas. Sin embargo, pueden indicar las configuraciones de servidor que pueden dar lugar a un bajo rendimiento, confiabilidad insuficiente, conflictos inesperados, mayores riesgos de seguridad u otros problemas potenciales.  
  
Best Practices Analyzer examina su equipo mediante reglas basadas en estos procedimientos recomendados y notifica los resultados. Cada regla de prácticas recomendadas incluye detalles sobre cómo cumplir la regla. Esta sección se documenta estas reglas para ayudarle a comprender las prácticas recomendadas que se aplican a Hyper-V, también interpretar los resultados cuando un examen encuentra las condiciones que no cumplen con los procedimientos recomendados.  
  
Para obtener más información sobre análisis y el analizador de procedimientos recomendados, consulte [Best Practices Analyzer](https://go.microsoft.com/fwlink/?LinkId=122786).  
  
## <a name="about-hyper-v"></a>Acerca de Hyper-V  
Hyper-V permite ejecutar varios sistemas operativos al mismo tiempo en un equipo físico, ejecutando cada uno de ellos en una máquina virtual. Las máquinas virtuales pueden ayudarle a usar sus recursos informáticos de forma más eficaz y flexible. Para obtener más información sobre Hyper-V, consulte [Hyper-V en Windows Server 2016](../Hyper-V-on-Windows-Server.md).
