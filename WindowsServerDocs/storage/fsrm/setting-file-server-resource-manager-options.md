# <a name="setting-file-server-resource-manager-options"></a>Setting File Server Resource Manager Options

> Se aplica a: Windows Server 2019, Windows Server 2016, Windows Server (canal semianual), Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2

Las opciones generales del servidor de archivos Administrador de recursos se pueden establecer en el cuadro de diálogo **Opciones de administrador de recursos del servidor de archivos** . Estos valores de configuración se usan en todos los nodos y algunos de ellos se pueden modificar cuando se trabaja con cuotas, se filtran archivos o se generan informes de almacenamiento.

Esta sección contiene los siguientes temas:

-   [Configurar notificaciones por correo electrónico](configure-email-notifications.md)
-   [Configurar límites de notificación](configure-notification-limits.md)
-   [Configurar informes de almacenamiento](configure-storage-reports.md)
-   [Configurar Auditoría de filtrado de archivos](configure-file-screen-audit.md)
