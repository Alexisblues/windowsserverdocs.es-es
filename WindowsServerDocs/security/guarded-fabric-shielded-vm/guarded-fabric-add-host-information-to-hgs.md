# <a name="add-host-information-to-the-hgs-configuration"></a>Agregue información del host a la configuración de HGS

>Se aplica a: Windows Server 2019, Windows Server (canal semianual), Windows Server 2016

Obtenga información acerca de los hosts desde el administrador del tejido y agregarlo a HGS. Este paso varía en función del modo de atestación:

- [Agregue información del host de atestación de administrador de confianza](guarded-fabric-add-host-information-for-admin-trusted-attestation.md)
- [Agregue información del host de atestación de TPM de confianza](guarded-fabric-add-host-information-for-tpm-trusted-attestation.md)
