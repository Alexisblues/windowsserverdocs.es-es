# <a name="remote-access-server-role-documentation"></a>Documentación de rol de servidor de acceso remoto

>Se aplica a: Windows Server (canal semianual), Windows Server 2016

Documentación de rol de servidor de acceso remoto proporciona instrucciones para implementar cualquiera de los tres servicios de rol (VPN de Always On, enrutamiento y servicio de acceso remoto, el Proxy de aplicación Web), individualmente o en el mismo servidor. Por ejemplo, estos documentos se aplican a situaciones donde ha implementado cualquier combinación de los tres servicios de rol, como la implementación de RRAS y DirectAccess en el mismo servidor.
