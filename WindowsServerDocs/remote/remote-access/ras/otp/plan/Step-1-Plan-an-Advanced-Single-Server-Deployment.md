# <a name="step-1-plan-an-advanced-single-server-deployment"></a>Paso 1 planear una implementación avanzada

>Se aplica a: Windows Server (canal semianual), Windows Server 2016

Planear y configurar una implementación avanzada del servidor único es el primer paso para planear un acceso remoto con la implementación de cliente de autenticación de contraseña de un solo uso (OTP).  
  
## <a name="plan-a-single-server-deployment"></a>Planear una implementación de servidor único  
Antes de implementar el acceso remoto con OTP, asegúrese de que ha completado todos los pasos para implementar un único servidor de acceso remoto. Consulte [implementar un único servidor de DirectAccess con configuración avanzada](https://technet.microsoft.com/windows-server-docs/networking/remote-access/directaccess/single-server-advanced/deploy-a-single-directaccess-server-with-advanced-settings).
