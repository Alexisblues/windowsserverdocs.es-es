# <a name="supported-remote-desktop-rdp-file-settings"></a>Admite la configuración de archivo RDP de escritorio remoto

En la tabla siguiente incluye la lista de configuración del archivo RDP compatibles que puede usar con los clientes de Windows y el código HTML. Una "x" en la columna de plataforma indica que se admite la configuración. Sin embargo, esta lista no es una lista completa de las configuraciones admitidas para los clientes de Windows y HTML5. Seguimos actualizar esta tabla para incluir más compatibles configuración de RDP para los clientes de Windows y HTML5, así como el macOS, iOS y Android clientes.

Para especificar un shell alternativo, escriba una ruta de acceso válida a un archivo ejecutable para el valor, como "C:\ProgramFiles\Office\word.exe".
