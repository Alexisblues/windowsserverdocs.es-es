# <a name="remote-desktop-services-hosting-partners-and-assessment"></a>Evaluación y socios de hospedaje de servicios de escritorio remoto

Recientemente, Microsoft entrega una nueva ruta de aprendizaje dentro de Microsoft Partner Network: "Hospedaje de escritorio de Windows y las aplicaciones que usan servicios de escritorio remoto en Azure".

Si es un partner de Microsoft y desean incluir en la lista de socios que han superado la evaluación, estos son los pasos que puede tardar en completarse la ruta de aprendizaje:

1. Convertirse en un [Microsoft Partner](https://partner.microsoft.com/), si no aún está.
2. Inspección del [hospedaje de Windows y las aplicaciones que usan servicios de escritorio remoto en la sesión de formación en Azure](https://mspartnerlp.partner.microsoft.com/LearningPath/LearningPath/DLPaths?trackId=2915&rowId=3603).
3. Tome el [evaluación técnica](https://mspartnerlp.partner.microsoft.com/LearningPath/LearningPath/DLPaths?trackId=1660&rowId=2220&trackPathId=9871).
4. Asegúrese de cumplir el [requisitos para la competencia Cloud Platform](https://partner.microsoft.com/en-us/membership/cloud-platform-competency).

¿Ya es un Partner de Microsoft y tiene alguna pregunta? Póngase en contacto con el equipo de escritorio remoto en <rdhostingpartners@microsoft.com>.  


## <a name="partners-who-have-passed-the-learning-path-assessment"></a>Los socios que han superado el aprendizaje de evaluación de la ruta de acceso 

Si es un cliente busca un socio que le ayudarán a aplicaciones en Azure para los usuarios y equipos de escritorio de Windows de host, hemos recopilado una lista de socios que han superado la evaluación. Este es un [lista de esos socios](https://github.com/MicrosoftDocs/windowsserverdocs/blob/master/WindowsServerDocs/remote/remote-desktop-services/RDS-Hosting-Partners.pdf), como del 28/03/2017, que puede descargar.

Puede encontrar más información sobre cada socio mediante estos pasos:

1. Abra [encuentre un socio](https://partnercenter.microsoft.com/pcv/search).
2. Desactive el **ubicación** campo.
3. Escriba el nombre del asociado en el **que estoy buscando ayuda con** campo.
