# <a name="multipoint-services"></a>MultiPoint Services
MultiPoint Services es una solución que permite que varios usuarios, cada uno con su propia y experimentan de Windows familiares, para compartir un equipo de manera simultánea.

Las estaciones de usuario, que constan de un monitor, un teclado y un mouse, están conectadas directamente con el equipo host a través de cables USB o de vídeo. Dado que MultiPoint Services es un producto de software publicado original de Microsoft, cuando tiene una licencia auténtica, eres elegible para recibir soporte técnico de Microsoft o un socio autorizado. Esto le ofrece las funcionalidades completas de Windows, el acceso a todas las actualizaciones más recientes y confianza que ha obtenido la experiencia esperada.

Dado que MultiPoint Services permite que varios usuarios compartan un equipo, puede proporcionar una alternativa de bajo costo para escenarios de informáticas tradicionales, donde cada usuario tiene su propio equipo. MultiPoint Services también proporciona una solución de fácil administración para la administración del sistema MultiPoint Services denominado MultiPoint Manager y una solución de administración fácil para el día\-a\-administración día, conocido como MultiPoint Panel.
