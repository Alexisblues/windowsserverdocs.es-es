# <a name="planning-a-multipoint-services-deployment"></a>Planeación de una implementación de MultiPoint Services
MultiPoint Services permite que varias estaciones se conecten a un equipo. En este caso varios usuarios pueden compartir un equipo al mismo tiempo. Cada estación consta de un concentrador de estaciones, monitor, teclado y mouse (ratón). MultiPoint Services incluye la aplicación MultiPoint Manager, que le ayuda, como un usuario administrativo, para supervisar y administrar estaciones MultiPoint y la aplicación de panel de MultiPoint, que proporciona funciones administrativas rutinarias.
