# <a name="enable-file-sharing-in-multipoint-services"></a>Habilitar el uso compartido de archivos en MultiPoint Services
Puede permitir que a los usuarios en sus estaciones MultiPoint para compartir archivos de dos maneras:  
  
-   **Si tiene un servidor de archivos en la red,** , se recomienda que cree una carpeta compartida del servidor de archivos.  
  
-   **Si tiene una pequeña red de servidores MultiPoint de 2-3, con ningún servidor de archivos dedicado,** uno de los servidores MultiPoint puede actuar como servidor de archivos para todos los equipos restantes que ejecuta MultiPoint services. Crear una carpeta compartida en ese servidor y, a continuación, crear cuentas de usuario local para todos los usuarios en ese servidor. La carpeta compartida puede estar en la unidad interna original, o puede asociar otras unidades internas o externas en el equipo.
