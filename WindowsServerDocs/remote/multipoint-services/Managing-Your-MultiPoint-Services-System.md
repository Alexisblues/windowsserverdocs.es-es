# <a name="managing-your-multipoint-services-system"></a>Administración del sistema MultiPoint Services
MultiPoint Services permite que varias estaciones se conecten a un equipo. Cada estación al uso consta de un [*concentrador de estaciones*](Switch-Between-Modes.md) o cliente Zero, un monitor, un teclado y un mouse. También se admiten los clientes conectados en red de protocolo de escritorio remoto (RDP).  
  
En esta imagen se muestra un diseño de ejemplo de un sistema MultiPoint Services con cuatro *estaciones*. Dicha configuración permite que varios usuarios usen el equipo al mismo tiempo y que realicen un trabajo independiente o una actividad de grupo.  
  
![Imagen del diseño del sistema MultiPoint Services](./media/WMSMultiPointServerSystemLayout.gif)  
  
MultiPoint Services incluye MultiPoint Manager, que le ayuda, como un usuario administrativo, supervisar y administrar el sistema MultiPoint, y el panel de MultiPoint, que proporciona funciones administrativas rutinarias. Los temas incluidos en esta guía de ayuda se describen muchas de las tareas que puede realizar en MultiPoint Manager y el panel de MultiPoint.
