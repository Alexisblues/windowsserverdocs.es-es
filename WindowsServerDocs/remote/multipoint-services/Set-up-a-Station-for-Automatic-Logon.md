# <a name="set-up-a-station-for-automatic-logon"></a>Configurar una estación de inicio de sesión automático
Inicio de sesión automático permite cada estación para iniciar sesión automáticamente cuando se inicia el equipo que está ejecutando MultiPoint Services y mostrar el escritorio. Un usuario administrativo puede establecer esta característica para estaciones individuales o para todas las estaciones.  
  
1.  Abra MultiPoint Manager en modo de estación y, a continuación, haga clic en el **estaciones** ficha.  
  
2.  Haga clic en el nombre de la estación que quiera que inicie sesión automáticamente.  
  
3.  En **Tareas**, haga clic en **Configurar estación**. Se abre la página **Configurar estaciones**.  
  
4.  Seleccione **inicio de sesión automático con la información siguiente**y, a continuación, escriba un **cuenta de usuario** nombre.  
  
5.  Escriba la contraseña de la cuenta de usuario y, a continuación, vuelva a escribir la contraseña para confirmarla.  
  
6.  Haga clic en **Aceptar**. Se cierra la página. El nombre de cuenta se muestra en el **inicio de sesión automático** columna.
