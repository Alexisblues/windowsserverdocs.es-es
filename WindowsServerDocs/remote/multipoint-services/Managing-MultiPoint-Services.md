# <a name="managing-multipoint-services"></a>Administrar MultiPoint Services
MultiPoint™ Server permite que varios usuarios, cada uno con su propia experiencia independiente de Windows, compartan un equipo al mismo tiempo. Las estaciones de usuario, que constan de un monitor, un teclado y un mouse, están conectadas directamente con el equipo host a través de cables USB o de vídeo o de la red.  
  
Use la información siguiente para obtener información acerca de las tareas que puede realizar en MultiPoint Manager y en el panel de MultiPoint, como administración de estaciones de MultiPoint Services mediante MultiPoint Manager y cómo usar el panel de MultiPoint diariamente.
