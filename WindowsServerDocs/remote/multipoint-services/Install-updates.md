# <a name="install-updates"></a>Instalar actualizaciones
Se recomienda que instale las actualizaciones si está disponible. Instalación de actualizaciones, requiere una conexión a Internet.  

1.  En la pantalla **Inicio**, abra el **Panel de control**.  
  
2.  En el Panel de Control, escriba **actualizaciones**y, a continuación, haga clic en **buscar actualizaciones**.  
  
3.  Si el sitio Web Windows Update enumera las actualizaciones que son necesarios en el equipo, instale las actualizaciones.
