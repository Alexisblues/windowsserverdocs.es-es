# <a name="install-updates"></a>Instalar actualizaciones
Se recomienda instalar las actualizaciones si está disponible. La instalación de actualizaciones requiere una conexión a Internet.  

1.  En la pantalla **Inicio**, abra el **Panel de control**.  
  
2.  En el panel de control, escriba **actualizaciones**y, a continuación, haga clic en **Buscar actualizaciones**.  
  
3.  Si el sitio web de Windows Update muestra las actualizaciones necesarias en el equipo, instale las actualizaciones.
