# <a name="important-information-about-software-license-compliance"></a>Información importante sobre el cumplimiento de la licencia de software
Debe cumplir los términos de licencia para los programas que usa en el sistema MultiPoint Services. También debe tener el número apropiado de licencias para el número de *estaciones* que agregue al sistema MultiPoint Services, tal como se describe en las secciones siguientes.  
  
## <a name="software-license-compliance"></a>Cumplimiento de licencias de software  
Si utiliza otros programas de Microsoft o los programas de terceros con este servicio o software de Microsoft, los términos que vienen con dichos programas se aplican al uso de ellos. Microsoft considere los errores para cumplir con estos términos también sean una infracción de la licencia de Microsoft.  
  
## <a name="client-access-license-cal-compliance"></a>Cumplimiento de licencias (CAL) de acceso de cliente  
Si desea agregar una estación a su sistema MultiPoint Services, debe proteger el número apropiado de *licencias de acceso de cliente (CAL)* para todas las estaciones conectadas a los servicios de MultiPoint.
