# <a name="manage-multipoint-systems-using-multipoint-dashboard"></a>Administrar sistemas MultiPoint mediante MultiPoint Dashboard
En Multipoint Dashboard, en la pestaña **sistemas** , puede hacer lo siguiente:

- Reiniciar o apagar sistemas seleccionados puede reiniciar o apagar los sistemas Multipoint Services seleccionados. Para más información sobre cómo reiniciar o apagar sistemas seleccionados, vea el tema [Reiniciar o apagar sistemas MultiPoint](Restart-or-Shut-Down-MultiPoint-Systems.md).
- Reasignar los sistemas seleccionados puede reasignar los sistemas Multipoint Services seleccionados. Para más información sobre cómo reasignar sistemas seleccionados, vea el tema [Reasignar los sistemas MultiPoint seleccionados](Remap-Selected-MultiPoint-Systems.md).
