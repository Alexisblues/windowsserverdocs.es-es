# <a name="vol"></a>vol



Muestra la etiqueta de volumen de disco y el número de serie, si existen.  Si se utiliza sin parámetros, **vol** muestra información de la unidad actual.

## <a name="syntax"></a>Sintaxis

```
vol [<Drive>:]
```

## <a name="parameters"></a>Parámetros

|Parámetro|Descripción|
|---------|-----------|
|\<Drive>:|Especifica la unidad que contiene el disco para el que desea mostrar la etiqueta de volumen y el número de serie.|
|/?|Muestra la ayuda en el símbolo del sistema.|
