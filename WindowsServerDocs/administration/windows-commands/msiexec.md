# <a name="msiexec"></a>msiexec



## <a name="msiexec"></a>Msiexec

Proporciona los medios para instalar, modificar y realizar operaciones en Windows Installer desde la línea de comandos.

Para obtener la sintaxis y ejemplos de cómo utilizar este comando, vea [msiexec](https://go.microsoft.com/fwlink/?LinkId=94329).
