# <a name="vssadmin"></a>vssadmin

>Se aplica a: Windows 10, Windows 8.1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, Windows Server 2008 R2, Windows Server 2008

Muestra las copias de seguridad copia sombra de volumen actual y todos los escritores de instantáneas instalados y proveedores. Seleccione un nombre de comando de la siguiente tabla de vista su sintaxis de comando.

|Comando|Descripción|
|---|---|
|[Vssadmin agregar shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788051(v%3dws.11))|Agrega una asociación de almacenamiento de copia de instantáneas de volumen.|
|[Vssadmin crear instantáneas](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788055(v%3dws.11))|Crea una nueva instantánea.|
|[Vssadmin delete sombras](vssadmin-delete-shadows.md)|Elimina las instantáneas de volumen.|
|[Vssadmin delete shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc785461(v%3dws.11))|Elimina las asociaciones de almacenamiento de instantáneas de volumen.|
|[Proveedores de lista vssadmin](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788108(v%3dws.11))|Las listas de registrar los proveedores de instantáneas de volumen.|
|[Sombras de lista vssadmin](vssadmin-list-shadows.md)|Listas existentes de instantáneas de volumen.|
|[Vssadmin lista shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788045(v%3dws.11))|Enumera todas las asociaciones de instantáneas en el sistema.|
|[Volúmenes de lista vssadmin](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788064(v%3dws.11))|Muestra los volúmenes que son aptos para instantáneas.|
|[Escritores de lista vssadmin](vssadmin-list-writers.md)|Enumera todos los volúmenes suscritos los escritores de instantáneas en el sistema.|
|[Vssadmin resize shadowstorage](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-server-2012-r2-and-2012/cc788050(v%3dws.11))|Cambia el tamaño máximo de una asociación de almacenamiento de instantáneas.|
