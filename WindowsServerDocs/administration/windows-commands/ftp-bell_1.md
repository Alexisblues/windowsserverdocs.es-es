# <a name="ftp-bell1"></a>FTP: bell_1

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Alterna un sonido que se produzca una vez completada cada comando de transferencia de archivos.   
## <a name="syntax"></a>Sintaxis  
```  
bell  
```  
### <a name="parameters"></a>Parámetros  
ninguno  
## <a name="remarks"></a>Comentarios  
De forma predeterminada, bell está desactivada.  
## <a name="BKMK_Examples"></a>Ejemplos  
Activar o desactivar un sonido que se produzca una vez completada cada comando de transferencia de archivos.  
```  
bell  
```
