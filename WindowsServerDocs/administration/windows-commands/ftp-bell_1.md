# <a name="ftp-bell"></a>ftp bell

> Se aplica a: Windows Server (canal semianual), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2 y Windows Server 2012

Alterna un sonido audible para que se produzca después de que se complete cada comando de transferencia de archivos. De forma predeterminada, este comando está desactivado.

## <a name="syntax"></a>Sintaxis

```
bell
```

### <a name="examples"></a>Ejemplos

Para alternar un sonido audible para que se produzca después de que se complete cada comando de transferencia de archivos, escriba:

```
bell
```
