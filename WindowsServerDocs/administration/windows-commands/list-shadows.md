# <a name="list-shadows"></a>sombras de lista



Enumera las persistente y existente no persistentes instantáneas que se encuentran en el sistema.

## <a name="syntax"></a>Sintaxis

```
list shadows {all | set <SetID> | id <ShadowID>}
```

## <a name="parameters"></a>Parámetros

|Parámetro|Descripción|
|---------|-----------|
|all|Enumera todas las instantáneas.|
|establecer \<SetID >|Enumera las instantáneas que pertenecen al identificador especificado de conjunto de copia sombra.|
|Id. de \<IdDeInstantánea >|Enumera las instantáneas con el identificador de copia de instantáneas especificada.|
