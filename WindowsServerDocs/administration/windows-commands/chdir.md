# <a name="chdir"></a>chdir

> Se aplica a: Windows Server (canal semianual), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2 y Windows Server 2012

Muestra el nombre del directorio actual o cambia el directorio actual. Si se usa solo con una letra de unidad (por `chdir C:`ejemplo,), **chdir** muestra los nombres del directorio actual en la unidad especificada. Si se usa sin parámetros, **chdir** muestra la unidad y el directorio actuales.

> > [!NOTE]
> Este comando se ha reemplazado con el **comando CD**. Para obtener más información, incluidos los detalles de la sintaxis y los parámetros, vea [CD (comando](cd.md)).
