# <a name="filesystems"></a>Sistemas de archivos



Muestra información sobre el sistema de archivos actual del volumen que tiene el foco y se enumeran los sistemas de archivos que se admiten para dar formato al volumen.

## <a name="syntax"></a>Sintaxis

```
filesystems
```

## <a name="remarks"></a>Comentarios

-   Debe seleccionarse un volumen para que esta operación se realice correctamente. Use la **seleccione volumen** comando para seleccionar un volumen y desplace el foco a ella.
