# <a name="ftp-close1"></a>FTP: close_1

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Finaliza la sesión ftp con el servidor remoto y sigue siendo el **ftp >** símbolo del sistema.   
## <a name="syntax"></a>Sintaxis  
```  
close  
```  
### <a name="parameters"></a>Parámetros  
ninguno  
## <a name="BKMK_Examples"></a>Ejemplos  
Finalizar la sesión ftp con el servidor remoto y permanecer en el **ftp >** símbolo del sistema.  
```  
close  
```
