# <a name="ftp-close_1"></a>FTP: close_1

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Finaliza la sesión FTP con el servidor remoto y permanece en el símbolo del sistema de **> FTP** .   
## <a name="syntax"></a>Sintaxis  
```  
close  
```  
#### <a name="parameters"></a>Parámetros  
ninguno  
## <a name="examples"></a><a name=BKMK_Examples></a>Example  
Finalice la sesión FTP con el servidor remoto y permanezca en el símbolo del sistema de **> FTP** .  
```  
close  
```
