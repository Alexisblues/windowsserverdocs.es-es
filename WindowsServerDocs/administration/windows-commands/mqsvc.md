# <a name="mqsvc"></a>mqsvc



La tecnología de Message Queue Server permite a las aplicaciones que se ejecutan en momentos diferentes comunicarse a través de redes y sistemas heterogéneos que pueden estar sin conexión temporalmente. Message Queue Server proporciona garantía de entrega de mensajes, enrutamiento eficaz, seguridad y mensajería por prioridades. Se puede usar para implementar soluciones para escenarios de mensajería asincrónicos y sincrónicos. Para obtener más información acerca de este comando, vea [Message Queue Server (MSMQ)](https://go.microsoft.com/fwlink/?LinkId=248723) en MSDN.

## <a name="syntax"></a>Sintaxis

```
Mqsvc.exe
```

#### <a name="parameters"></a>Parámetros

None
