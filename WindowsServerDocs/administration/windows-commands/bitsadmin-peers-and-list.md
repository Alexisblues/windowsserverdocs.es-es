# <a name="bitsadmin-peers-and-list"></a>Bitsadmin elementos del mismo nivel y lista



Enumera todos los elementos del mismo nivel.

## <a name="syntax"></a>Sintaxis

```
bitsadmin /Peers /List 
```

## <a name="BKMK_examples"></a>Ejemplos

El ejemplo siguiente enumeran todos los elementos del mismo nivel.
```
C:\>bitsadmin /Peers /List
```
