# <a name="help"></a>ayuda

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Muestra una lista de los comandos disponibles o la información de ayuda detallada acerca de un comando especificado.  
  
  
  
## <a name="syntax"></a>Sintaxis  
  
```  
help [<command>]  
```  
  
## <a name="parameters"></a>Parámetros  
  
| Parámetro |                              Descripción                              |
|-----------|-----------------------------------------------------------------------|
| <command> | Especifica el comando que se va a mostrar información de ayuda detallada. |
  
## <a name="remarks"></a>Comentarios  
  
-   Si no se especifica ningún comando, **ayuda** mostrará todos los comandos posibles.  
  
## <a name="BKMK_examples"></a>Ejemplos  
Para mostrar una lista de todos los comandos disponibles en DiskPart, escriba:  
  
```  
help  
```  
  
Para mostrar información de ayuda detallada acerca de cómo usar el **crear partición principal** comando DiskPart, escriba:  
  
```  
help create partition primary  
```
