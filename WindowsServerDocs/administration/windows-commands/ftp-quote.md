# <a name="ftp-quote"></a>FTP: oferta

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Argumentos textuales se envía al servidor ftp remoto. Se devuelve un código de respuesta única de ftp.   
## <a name="syntax"></a>Sintaxis  
```  
quote <Argument>[ ]  
```  
### <a name="parameters"></a>Parámetros  
|Parámetro|Descripción|  
|-------|--------|  
|<Argument>|Especifica el argumento que se enviará al servidor ftp.|  
## <a name="remarks"></a>Comentarios  
El **oferta** comando es idéntico a la **literal** comando.  
## <a name="BKMK_Examples"></a>Ejemplos  
Enviar un **salga** comando al servidor ftp remoto.  
```  
quote quit  
```
