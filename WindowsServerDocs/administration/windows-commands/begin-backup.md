# <a name="begin-backup"></a>Comenzar la copia de seguridad



Inicia una sesión de copia de seguridad completa.

## <a name="syntax"></a>Sintaxis

```
begin backup
```

## <a name="remarks"></a>Comentarios

-   Este comando reemplaza la configuración de copia de seguridad de copia de forma predeterminada.
