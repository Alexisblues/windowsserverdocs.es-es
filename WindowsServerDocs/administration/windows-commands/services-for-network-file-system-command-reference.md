# <a name="services-for-network-file-system-command-reference"></a>Servicios de referencia de comandos de sistema de archivos de red



Servicios para Network File System (NFS) proporciona una solución que permite transferir archivos entre equipos que ejecutan Windows Server 2008 y sistemas operativos UNIX mediante el protocolo NFS para compartir de archivos.

La siguiente es una lista de herramientas de línea de comandos de NFS.

|Comando|Descripción|
|-------|-----------|
|[mapadmin](mapadmin.md)|Administrar la asignación para los servicios de Microsoft para Network File System de nombre de usuario.|
|[montaje](mount.md)|Recursos compartidos de red de Network File System (NFS) de montaje.|
|[Nfsadmin](nfsadmin.md)|Administrar servidor para NFS y cliente para NFS.|
|[Nfsshare](nfsshare.md)|Controlar los recursos compartidos de Network File System (NFS).|
|[Nfsstat](nfsstat.md)|Mostrar o restablecer recuentos de llamadas realizadas al servidor para NFS.|
|[Rpcinfo](rpcinfo.md)|Lista de programas en equipos remotos.|
|[Showmount](showmount.md)|Mostrar directorios montados.|
