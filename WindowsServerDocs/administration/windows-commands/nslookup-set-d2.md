# <a name="nslookup-set-d2"></a>nslookup set d2



Activa o desactiva el modo de depuración exhaustiva. Se imprimen todos los campos de cada paquete.

## <a name="syntax"></a>Sintaxis

```
set [no]d2
```

## <a name="parameters"></a>Parámetros

| Parámetro |                             Descripción                              |
|-----------|----------------------------------------------------------------------|
| **nod2**  | Desactiva el modo de depuración exhaustiva. La sintaxis predeterminada es **NOD2**. |
|  **D2**   |                 Activa el modo de depuración exhaustiva.                  |
|   {ayuda   |                                  ?}                                  |
