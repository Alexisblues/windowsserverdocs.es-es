# <a name="nslookup-set-d2"></a>nslookup set d2



Activa o desactiva la exhaustiva de modo de depuración. Se imprimen todos los campos de todos los paquetes.

## <a name="syntax"></a>Sintaxis

```
set [no]d2
```

## <a name="parameters"></a>Parámetros

|Parámetro|Descripción|
|---------|-----------|
|**nod2**|Desactiva el modo de depuración exhaustiva. La sintaxis predeterminada es **nod2**.|
|**d2**|Activa el modo de depuración exhaustiva.|
|{Ayuda | ?}|Muestra un resumen breve de **nslookup** subcomandos.|
