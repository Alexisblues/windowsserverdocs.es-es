# <a name="bitsadmin-help"></a>bitsadmin help



Muestra el uso de la línea de comandos.

## <a name="syntax"></a>Sintaxis

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>Example

En el ejemplo siguiente se recupera la ayuda de la línea de comandos.
```
C:\>bitsadmin /help
```
