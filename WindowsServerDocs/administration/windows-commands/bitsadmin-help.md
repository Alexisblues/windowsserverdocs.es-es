# <a name="bitsadmin-help"></a>bitsadmin help



Muestra el uso de la línea de comandos.

## <a name="syntax"></a>Sintaxis

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>Ejemplos

El ejemplo siguiente recupera de la Ayuda de línea de comandos.
```
C:\>bitsadmin /help
```
