# <a name="ftp-glob1"></a>ftp: glob_1

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Alterna la expansión de caracteres comodín de los nombres de archivo local.   
## <a name="syntax"></a>Sintaxis  
```  
glob  
```  
### <a name="parameters"></a>Parámetros  
ninguno  
## <a name="remarks"></a>Comentarios  
- De forma predeterminada, uso de comodines está activado.  
- Esta opción permite el uso del signo de interrogación (?) y asterisco (*) como caracteres comodín en nombres de archivo o ruta de acceso locales.  
  ## <a name="BKMK_Examples"></a>Ejemplos  
  Alternar expansión de caracteres comodín de los nombres de archivo local.  
  ```  
  glob  
  ```
