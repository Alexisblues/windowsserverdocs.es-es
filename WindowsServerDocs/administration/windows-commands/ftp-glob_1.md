# <a name="ftp-glob_1"></a>FTP: glob_1

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Alterna la expansión de caracteres comodín de los nombres de archivos locales.   
## <a name="syntax"></a>Sintaxis  
```  
glob  
```  
#### <a name="parameters"></a>Parámetros  
ninguno  
## <a name="remarks"></a>Comentarios  
- De forma predeterminada, comodines está activado.  
- Comodines permite el uso del asterisco (*) y el signo de interrogación (?) como caracteres comodín en los nombres de archivo o ruta de acceso local.  
  ## <a name="examples"></a><a name=BKMK_Examples></a>Example  
  Alternar expansión de caracteres comodín de los nombres de archivos locales.  
  ```  
  glob  
  ```
