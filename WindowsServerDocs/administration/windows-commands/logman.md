# <a name="logman"></a>logman

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

**Logman** crea y administra registros de rendimiento y sesión de seguimiento de eventos y admite muchas funciones del monitor de rendimiento desde la línea de comandos.
## <a name="syntax"></a>Sintaxis
```
logman [create | query | start | stop | delete| update | import | export | /?] [options]
```
## <a name="actions"></a>Acciones
|.|Descripción|
|-----|--------|
|[logman create](logman-create.md)|Cree un contador, un seguimiento, un recopilador de datos de configuración o una API.|
|[logman query](logman-query.md)|propiedades del recopilador de datos de consulta.|
|[logman start &#124; stop](logman-start-stop.md)|iniciar o detener la recopilación de datos.|
|[logman delete](logman-delete.md)|eliminar un recopilador de datos existente.|
|[logman update](logman-update.md)|Actualice las propiedades de un recopilador de datos existente.|
|[logman import &#124; export](logman-import-export.md)|importar un conjunto de recopiladores de datos de un archivo XML o exportar un conjunto de recopiladores de datos a un archivo XML.|
