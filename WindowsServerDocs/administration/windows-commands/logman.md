# <a name="logman"></a>logman

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

**logman** crea y administra los registros de sesión de seguimiento de eventos y el rendimiento y es compatible con muchas funciones del Monitor de rendimiento desde la línea de comandos.
## <a name="syntax"></a>Sintaxis
```
logman [create | query | start | stop | delete| update | import | export | /?] [options]
```
## <a name="actions"></a>Acciones
|Acción|Descripción|
|-----|--------|
|[crear Logman](logman-create.md)|crear un contador, seguimiento, el recopilador de datos de configuración o API.|
|[logman consulta](logman-query.md)|propiedades del recopilador de datos de consulta.|
|[logman start &#124; detener](logman-start-stop.md)|iniciar o detener la recopilación de datos.|
|[logman delete](logman-delete.md)|eliminar un recopilador de datos existente.|
|[logman update](logman-update.md)|Actualizar las propiedades de un recopilador de datos existente.|
|[logman import &#124; exportar](logman-import-export.md)|importar un conjunto de recopiladores de datos desde un archivo XML o exportar un recopilador de datos establecido en un archivo XML.|
