# <a name="bitsadmin-peers-and-help"></a>bitsadmin elementos del mismo nivel y ayuda

Muestra el uso de la línea de comandos para los modificadores de **/Peers** .

## <a name="syntax"></a>Sintaxis

```
bitsadmin /peers /help
```

## <a name="examples"></a><a name=BKMK_examples></a>Example

El siguiente comando muestra la ayuda de la línea de comandos para los modificadores **/Peers** .

```
C:\>bitsadmin /peers /help
```
