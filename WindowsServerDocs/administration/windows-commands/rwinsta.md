# <a name="rwinsta"></a>rwinsta

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Permite restablecer (eliminar) una sesión en un servidor Host de sesión de escritorio remoto (Host de sesión de rd).

> [!NOTE]
> En Windows Server 2008 R2, el nombre de Terminal Services se cambió a Servicios de Escritorio remoto. Para descubrir las novedades de la versión más reciente, consulte [Novedades en servicios de escritorio remoto en Windows Server 2012](https://technet.microsoft.com/library/hh831527) en la biblioteca de TechNet de Windows Server.

## <a name="remarks"></a>Comentarios
Este comando es el mismo que el **restablecer sesión** comando.
