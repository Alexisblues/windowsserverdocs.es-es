# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache y clear

Purga la memoria caché local.

## <a name="syntax"></a>Sintaxis

```
bitsadmin /cache /clear
```

## <a name="examples"></a>Ejemplos

Para purgar la memoria caché local:

```
bitsadmin /cache /clear
```
