# <a name="bitsadmin-cache-and-getexpirationtime"></a>getexpirationtime y bitsadmin caché



Recupera la hora de expiración de caché.

## <a name="syntax"></a>Sintaxis

```
bitsadmin /Cache /GetExpirationTime 
```

## <a name="BKMK_examples"></a>Ejemplos

El siguiente recupera la hora de expiración de caché.
```
C:\>bitsadmin /Cache /GetExpirationTime
```
