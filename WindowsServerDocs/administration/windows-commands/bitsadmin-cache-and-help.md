# <a name="bitsadmin-cache-and-help"></a>memoria caché y ayuda de bitsadmin

Muestra el uso de la línea de comandos para los modificadores de **caché** .

## <a name="syntax"></a>Sintaxis

```
bitsadmin /cache /help
```

## <a name="examples"></a><a name=BKMK_examples></a>Example

El siguiente comando muestra la ayuda de la línea de comandos para los modificadores de **caché** .

```
bitsadmin /cache /help
```
