# <a name="bitsadmin-cache-and-help"></a>Ayuda y bitsadmin caché



Muestra el uso de la línea de comandos para el **caché** conmutadores.

## <a name="syntax"></a>Sintaxis

```
Bitsadmin /Cache /Help 
```

## <a name="BKMK_examples"></a>Ejemplos

El comando siguiente muestra la Ayuda de línea de comandos para el **caché** conmutadores.
```
Bitsadmin /Cache /Help
```
