# <a name="logman-update"></a>logman update



Actualizar un recopilador de datos existente.

## <a name="syntax"></a>Sintaxis

```
logman update <counter | trace | alert | cfg | api> <[-n] <name>> [options]
```

### <a name="parameters"></a>Parámetros

|Parámetro|Descripción|
|---------|-----------|
|[Logman (actualizar contador)](logman-update-counter.md)|Actualizar un recopilador de datos de contador.|
|[Logman actualizar seguimiento](logman-update-trace.md)|Actualizar un recopilador de datos de seguimiento.|
|[alerta de actualización de Logman](logman-update-alert.md)|Actualizar un recopilador de datos de alertas.|
|[Logman Update cfg](logman-update-cfg.md)|Actualizar un recopilador de datos de configuración.|
|[Logman Update API](logman-update-api.md)|Actualizar un recopilador de datos de seguimiento de API.|
