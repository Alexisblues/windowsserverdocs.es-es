# <a name="bitsadmin-peercaching-and-help"></a>Ayuda y bitsadmin caché del mismo nivel



Muestra el uso de la línea de comandos para el **caché del mismo nivel** conmutadores.

## <a name="syntax"></a>Sintaxis

```
Bitsadmin /PeerCaching /Help 
```

El comando siguiente muestra la Ayuda de línea de comandos para el **caché del mismo nivel** conmutadores.
```
Bitsadmin /PeerCaching /Help
```
