# <a name="using-the-enable-command"></a>Usar el comando enable

> Se aplica a: Windows Server (canal semianual), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2 y Windows Server 2012

Habilita todos los servicios para servicios de implementación de Windows.

## <a name="subcommands"></a>Subcomandos
|Subommand|Descripción|
|-------|--------|
|[Usar el comando Enable-Server](using-the-enable-server-command.md)|Habilita todos los servicios en un servidor de servicios de implementación de Windows especificado (servidor de implementación).|
|[Usar el comando enable-TransportServer](using-the-enable-transportserver-command.md)|Habilita todos los servicios en un servidor de transporte especificado.|
