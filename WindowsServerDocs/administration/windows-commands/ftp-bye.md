# <a name="ftp-bye"></a>FTP: bye

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Finaliza la sesión ftp con el equipo remoto y se cierra.   
## <a name="syntax"></a>Sintaxis  
```  
bye  
```  
### <a name="parameters"></a>Parámetros  
ninguno  
## <a name="remarks"></a>Comentarios  
**Adiós** y **salga** ambos finalización la sesión de ftp con el equipo remoto y se cierra.  
## <a name="BKMK_Examples"></a>Ejemplos  
Finalizar la sesión ftp con el equipo remoto y salir.  
```  
bye  
```
