# <a name="bitsadmin-cache-and-getlimit"></a>getlimit y bitsadmin caché



Recupera el límite de caché.

## <a name="syntax"></a>Sintaxis

```
bitsadmin /Cache /GetLimit 
```

## <a name="BKMK_examples"></a>Ejemplos

El ejemplo siguiente recupera el límite de caché...
```
C:\>bitsadmin /Cache /GetLimit 
```
