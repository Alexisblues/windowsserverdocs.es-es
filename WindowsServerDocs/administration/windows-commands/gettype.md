# <a name="gettype"></a>gettype

El comando **GetType** está en desuso y no se garantiza que se admita en versiones futuras de Windows.

Esta herramienta se incluye en Windows Server 2003. Para obtener más información, vea [GetType](https://docs.microsoft.com/previous-versions/orphan-topics/ws.10/cc773104(v=ws.10)).
