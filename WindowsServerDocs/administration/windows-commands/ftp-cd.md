# <a name="ftp-cd"></a>FTP: CD

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

cambia el directorio de trabajo en el equipo remoto.   
## <a name="syntax"></a>Sintaxis  
```  
cd <remotedirectory>  
```  
#### <a name="parameters"></a>Parámetros  

|     Parámetro     |                                 Descripción                                 |
|-------------------|-----------------------------------------------------------------------------|
| <remotedirectory> | Especifica el directorio del equipo remoto al que desea cambiar. |

## <a name="examples"></a><a name=BKMK_Examples></a>Example  
Cambie el directorio del equipo remoto a **docs**.  
```  
cd Docs  
```  
Cambie el directorio del equipo remoto a **vídeos de mayo**.  
```  
cd  May Videos  
```
