# <a name="ftp-cd"></a>ftp: cd

>Se aplica a: Windows Server (canal semianual), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

cambia el directorio de trabajo en el equipo remoto.   
## <a name="syntax"></a>Sintaxis  
```  
cd <remotedirectory>  
```  
### <a name="parameters"></a>Parámetros  

|     Parámetro     |                                 Descripción                                 |
|-------------------|-----------------------------------------------------------------------------|
| <remotedirectory> | Especifica el directorio en el equipo remoto al que desea cambiar. |

## <a name="BKMK_Examples"></a>Ejemplos  
Cambie el directorio en el equipo remoto a **Docs**.  
```  
cd Docs  
```  
Cambie el directorio en el equipo remoto a **vídeos pueden**.  
```  
cd  May Videos  
```
