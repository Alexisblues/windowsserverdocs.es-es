# <a name="windows-server-technical-preview-expiration"></a>Expiración de Windows Server Technical Preview
Cada una de las versiones de Windows Server 2016 Technical Preview contiene tiempo\-software confidencial. A partir de la fecha de expiración, dejará de tener autorización para usar la versión.  

## <a name="license-expiration-dates-for-technical-previews"></a>Fechas de expiración de la licencia de vistas previas técnicas  
En esta tabla figura la fecha de expiración de la licencia de cada una de las versiones de Windows Server 2016 Technical Preview.  

|Publicación|Fecha de la versión|Fecha de expiración de la licencia|  
|-----------|----------------|---------------------------|  
|Windows Server "2016 Technical Preview 3|Agosto de 2015|1 de agosto de 2016|  
|Windows Server "2016 Technical Preview 4|Noviembre de 2015|15 de octubre de 2016|  
|Windows Server "2016 Technical Preview 5|Abril de 2016|28 de febrero de 2017|
