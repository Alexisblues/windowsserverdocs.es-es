# <a name="define-display-settings"></a>Definir la configuración de pantalla

>Se aplica a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Según el hardware que está conectado al servidor, se debe establecer la resolución de pantalla. Visualización óptima del panel, se necesita una configuración de 1024 x 768 o superior. Para obtener más información acerca de cómo establecer la resolución de pantalla, consulta el [Windows Assessment and Deployment Kit](https://go.microsoft.com/fwlink/?LinkId=248694) documentación.
