# <a name="customize-disk-partitions"></a>Personalización de las particiones de disco

>Se aplica a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Para personalizar la configuración de partición, puedes crear previamente la imagen mediante un archivo unattend.xml. Se recomienda encarecidamente que se usa una partición de datos dedicada en todas las configuraciones para garantizar que el servidor puede ser fácilmente la copia de seguridad y restaurar.
