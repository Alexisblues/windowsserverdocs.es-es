# <a name="get-started-with-windows-server-2016-essentials"></a>Introducción a Windows Server 2016 Essentials 

>Se aplica a: Windows Server 2016 Essentials

**Windows Server 2016 Essentials** edición está conectada a la nube primera server, ideal para pequeñas empresas con hasta 25 usuarios y dispositivos de 50. Essentials es una buena opción para los clientes que actualmente usan la edición Foundation, que no está disponible en Windows Server 2016. Para obtener más información, consulta [Novedades en Windows Server Essentials](what-s-new.md) y [descargue la evaluación de Windows Server Essentials](https://technet.microsoft.com/evalcenter/dn205288.aspx?wt.mc_id=TEC_144_1_7).
