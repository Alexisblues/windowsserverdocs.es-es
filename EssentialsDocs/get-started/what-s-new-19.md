#<a name="whats-new-in-windows-server-2019-essentials"></a>Novedades de Windows Server 2019 Essentials

> Se aplica a: Windows Server 2019 Essentials

## <a name="windows-server-essentials-experience-role-has-been-deprecated"></a>Rol de experiencia de Windows Server Essentials en desuso

Se quitó el rol de experiencia de Windows Server Essentials de todos los SKU de servidor, incluidos Windows Server Essentials de 2019. Esto significa que el [panel administrativo](../manage/overview-of-the-dashboard-in-windows-server-essentials.md) que solía ser la característica principal de rol de experiencia de Essentials ya no es accesible y toda la administración y configuración deben realizarse manualmente. 

Con la degradación de la función servidor de la experiencia de Essentials, ya no están disponibles las siguientes características:

-   Copia de seguridad de cliente 
-   Acceso web remoto 

## <a name="microsoft-365-business-the-modern-small-business-solution"></a>Microsoft 365 empresa: La solución moderna de pequeña empresa 

En los años desde el lanzamiento de Windows Server 2016 Essentials, nos hemos trabajado intensamente en la siguiente solución para grandes y pequeñas empresas: Microsoft 365. Se recomienda Microsoft 365 para clientes de pequeñas empresas como sustituto de todas las versiones de Windows Server Essentials. Microsoft 365 es una solución moderna para la colaboración y uso compartido de archivos e incluye Office 365, Windows 10 y Enterprise Mobility + Security. 

Desarrollar aún más su negocio con [Microsoft 365 empresa](https://www.microsoft.com/microsoft-365/business), que incluye más características de Office, como correo electrónico y calendario, el almacenamiento en la nube, protección de datos etc. de archivos. Microsoft 365 empresa le ayudará a de pequeñas empresas a alcanzar nuevos niveles de productividad.

## <a name="aad-connect-can-now-be-installed-on-windows-server-2019-essentials"></a>AAD Connect ahora puede instalarse en Windows Server Essentials de 2019

Windows Server Essentials de 2019 incluye nuevo soporte para Azure Active Directory a través de [AAD Connect](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-prerequisites). 

## <a name="ltsc-support-for-windows-server-2016-essentials"></a>Soporte técnico LTSC para Windows Server 2016 Essentials

Se admitirá a los clientes que actualmente usan Windows Server 2016 Essentials según el largo término mantenimiento canal (LTSC) mantenimiento de la línea de tiempo.
Para compañías con más de 25 dispositivos de usuarios/50, o las empresas que han crecido potencialmente más allá de 25 usuarios, Windows Server Standard 2019 – y potencialmente Datacenter edition: puede proporcionar opciones de implementación más flexibles.
