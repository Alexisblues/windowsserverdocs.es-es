# <a name="release-documentation-for-windows-server-essentials"></a>Documentación de la versión para Windows Server Essentials

>Se aplica a: Windows Server 2016 Essentials

Bienvenido a la documentación de la versión del software de servidor de Windows Server 2016 Essentials. Este tema contiene información sobre los problemas que pueden surgir al instalar y ejecutar el software de servidor de 2016 de Windows Server Essentials. Debe revisar este tema antes de instalar y ejecutar Windows Server 2016 Essentials de la red.  
  
## <a name="client-computers"></a>Equipos cliente  
  
### <a name="install-all-available-critical-and-security-updates-from-windows-update"></a>Instalar todas las actualizaciones críticas y de seguridad disponibles desde Windows Update  

Antes de conectar un equipo cliente al servidor, asegúrese de que abra Windows Update en el equipo cliente e instale todas las actualizaciones críticas y de seguridad disponibles.  
  
### <a name="client-computer-restore-may-not-succeed-after-migration-to-windows-10"></a>Restauración del equipo cliente podría no realizarse correctamente tras la migración a Windows 10  
 Después de actualizar un equipo cliente de Windows 8.1 a Windows 10, la restauración del equipo cliente no funcionen correctamente. Deberá volver a conectar al cliente de Windows Server 2016 Essentials.
