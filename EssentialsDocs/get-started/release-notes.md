# <a name="release-documentation-for-windows-server-essentials"></a>Documentación de la versión para Windows Server Essentials

>Se aplica a: Windows Server 2016 Essentials

Bienvenido a la documentación de la versión del software de servidor de Windows Server 2016 Essentials. Este tema contiene información sobre problemas que puedes encontrar al instalar y ejecutar el software de servidor de 2016 de Windows Server Essentials. Debes revisar este tema antes de instalar y ejecutar Windows Server 2016 Essentials en la red.  
  
## <a name="client-computers"></a>Equipos cliente  
  
### <a name="install-all-available-critical-and-security-updates-from-windows-update"></a>Instala todas las actualizaciones críticas y de seguridad disponibles desde Windows Update  

Antes de conectar un equipo cliente al servidor, asegúrate de que abre Windows Update en el equipo cliente e instalar todas las actualizaciones críticas y de seguridad disponibles.  
  
### <a name="client-computer-restore-may-not-succeed-after-migration-to-windows-10"></a>Restauración de equipo cliente no puede correctamente después de la migración a Windows 10  
 Después de actualizar un equipo cliente de Windows 8.1a Windows 10, restauración del equipo cliente no funcionen correctamente. Tendrás que volver a conectar al cliente de Windows Server 2016 Essentials.
