# <a name="use-the-my-server-app-to-connect-to-windows-server-essentials"></a>Usa la mi aplicación de Server para conectar con Windows Server Essentials

>Se aplica a: Windows Server 2016 Essentials, Windows Server 2012 R2 Essentials, Windows Server 2012 Essentials

Las aplicaciones mi servidor para Windows Server Essentials te permiten conectarte a los recursos y realizar tareas administrativas claras en el servidor de Windows Server Essentials desde tu portátil, equipo o dispositivo Surface. En mi servidor, puedes administrar los usuarios, dispositivos y alertas y trabajar con archivos compartidos en el servidor. Cuando estás sin conexión, puedes seguir trabajar con archivos que se ha accedido recientemente en mi servidor; los cambios sin conexión se sincronizan automáticamente con el servidor la próxima vez que te conectes.  
  
 Si el servidor ejecuta el sistema operativo de Windows Server Essentials, usa la aplicación de mi servidor original. Si el servidor está ejecutando el sistema operativo de Windows Server Essentials, debes usar la aplicación actualizada de mi Server 2012 R2. Ambas aplicaciones se pueden descargar desde [aplicaciones para Windows](https://windows.microsoft.com/windows-8/apps).  
  
> [!TIP]

>  Para acceder a recursos en Windows Server Essentials desde tu Windows Phone, usa la aplicación de teléfono de mi servidor (para Windows Server Essentials) o la aplicación de teléfono de mi Server 2012 R2 (para Windows Server Essentials). Para obtener información acerca de la aplicación de teléfono de mi servidor, consulte [usar la aplicación de mi servidor para Windows Phone](Work-Remotely-in-Windows-Server-Essentials.md#BKMK_2). Para obtener información acerca de la aplicación de teléfono de mi Server 2012 R2, consulta la entrada de blog [aplicaciones de Windows de mi Server 2012 R2 y Windows Phone](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx).

##  <a name="BKMK_WhatsNew"></a>¿Qué s nuevas en mi Server 2012 R2  
 Además de las características disponibles en el servidor de mi, mi Server 2012 R2 ofrece las siguientes características nuevas para clientes con Windows Server Essentials:  
  
-   **Acceso a sitios de SharePoint Online equipo y bibliotecas** -aprovechar Microsoft Office 365 integración con Windows Server Essentials para tener acceso a sitios de SharePoint Online equipo y abrir documentos en las bibliotecas de SharePoint Online en mi Server 2012 R2.  
  
-   **Conexiones a Escritorio remoto a los equipos servidor y cliente** -uso **conexión remota** en mi Server 2012 R2 para conectarse a los equipos cliente y servidor de Windows Server Essentials mediante Escritorio remoto.  
  
##  <a name="BKMK_OS"></a>Requisitos del sistema operativo  
 Para usar mi servidor o Mi Server 2012 R2 en tu portátil, equipo o dispositivo Surface, el equipo debe tener uno de los siguientes sistemas operativos: Windows 8.1, Windows RT 8.1, Windows 8 o Windows RT.  
  
##  <a name="BKMK_Install"></a>Instalar el servidor  
 Puedes descargar la aplicación de mi servidor correspondiente de la [aplicaciones para Windows](https://windows.microsoft.com/windows-8/apps) de la tienda. Si el equipo tiene el sistema operativo Windows 8 o Windows RT y desea conectarse al servidor de la intranet mediante el nombre del servidor, también debes instalar un certificado del servidor.  
  
#### <a name="to-install-my-server-or-my-server-2012-r2-on-your-windows-based-laptop-pc-or-surface-device"></a>Para instalar mi servidor o Mi Server 2012 R2 en tu portátil basado en Windows, el equipo o el dispositivo Surface  
  
1.  Instalar la aplicación de mi servidor correspondiente desde [aplicaciones para Windows](https://windows.microsoft.com/windows-8/apps) en tu portátil basado en Windows, PC o dispositivo Surface.  
  
    |Sistema operativo de servidor|Descargar|  
    |-----------------------------|--------------|  
    | Windows Server Essentials|[Mi servidor](https://apps.microsoft.com/webpdp/app/19d94f81-db21-4234-8b49-806694dbbaea)|  
    | Windows Server Essentials|[Mi Server 2012 R2](https://apps.microsoft.com/webpdp/app/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
  
2.  Si quieres poder usar el nombre del servidor para conectarse al servidor de Windows Server Essentials a través de una intranet y el dispositivo tiene el sistema operativo Windows 8 o Windows RT, instalar el certificado de CAROOT.cer de predeterminado en el equipo realizando el siguiente procedimiento. Instalación manual de un certificado no es necesario en Windows 8.1 y Windows RT 8.1.  
  
###  <a name="BKMK_InstallCert"></a>Para instalar un certificado para mi servidor en el dispositivo de Windows 8, Windows 8.1 o Windows RT  
  
1.  Abre un explorador web en el equipo y descargar el certificado de manera predeterminada, CAROOT.cer, desde el servidor. Para ello, escribe lo siguiente, sustituyendo el nombre del servidor (por ejemplo, **marketing.contoso.com**) para <*nombreDeServidor*>:  
  
     **http://<servername\>/Connect/default.aspx?Get=caroot.cer**  
  
2.  Instalar el certificado:  
  
    1.  En la **inicio** página abierta **Explorador de archivos**.  
  
    2.  Asegúrate de que se muestran los elementos ocultos y extensiones de nombre de archivo: en el **vista** pestaña la **mostrar u ocultar** grupo, selecciona el **elementos ocultos** y **extensiones de nombre de archivo** casillas de verificación.  
  
    3.  Navega a la carpeta que contiene el archivo CAROOT.cer que acabas de descargar.  
  
    4.  Haz clic en el archivo CAROOT.cer y, a continuación, haz clic en **abrir**.  
  
    5.  En la **certificado** cuadro de diálogo, haz clic en **instalar certificado**.  
  
    6.  Selecciona **máquina Local** como la ubicación de instalación y, a continuación, haz clic en **siguiente**.  
  
    7.  En la **almacén de certificados** página del asistente, selecciona **colocar todos los certificados en el siguiente almacén**y usar **examinar** para elegir el **entidades de certificación raíz de confianza** de la tienda. A continuación, haz clic en **finalizar**.

2.  Inicia sesión en el servidor de Windows Server Essentials mediante tu cuenta en el servidor. Para identificar el servidor:  
  
    -   En un entorno externo, usa el nombre de dominio del servidor s por ejemplo, **marketing.contoso.com**.  
  
    -   Si se conecta locales de intranet, usa el nombre del servidor en el ejemplo anterior, es el nombre del servidor **marketing**. (Si usas un equipo con Windows 8 o Windows RT para conectar local y que quieres usar este método, debes instalar un certificado de CAROOT.cer desde el servidor. Para obtener más información, consulta [para instalar un certificado para mi servidor en el dispositivo de Windows 8, Windows 8.1 o Windows RT](#BKMK_InstallCert).)

##  <a name="BKMK_ConnectServer"></a>Cómo conectarse a tu servidor de la red local  
 Para usar correctamente la aplicación del servidor mi en Windows Server Essentials para Windows Phone, Windows 8 y Windows 8.1, primero debes instalar el certificado de servidor en el dispositivo. Esto conecta el dispositivo al servidor que ejecuta Windows Server Essentials en la red local. Usa los siguientes procedimientos para conectarse al servidor en la red local e instalar el certificado de servidor en tu Windows Phone o en un equipo que ejecute Windows 8 o Windows 8.1.  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>Para conectarse al servidor de tu Windows Phone  
  
1.  En tu Windows Phone que ejecute Windows 8 o Windows 8.1, abre Internet Explorer.  
  
2.  En la barra de direcciones, escriba **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, y, a continuación, presione ENTRAR.  
  
3.  Cuando se te pida instalar el certificado caroot.cer, haz clic en **instalar**.  
  
4.  Tiempo de espera completar la instalación del certificado.  
  
    > [!NOTE]
    >  Una vez completada la instalación del certificado, iniciar sesión a la aplicación de mi servidor para Windows Phone con el nombre del servidor y las credenciales de red.  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>Para conectarse a su servidor en la red local desde un equipo que ejecute Windows 8 o Windows 8.1  
  
1.  En un equipo que ejecute Windows 8 o Windows 8.1, abre Internet Explorer.  
  
2.  En la barra de direcciones, escriba **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, y, a continuación, presione ENTRAR.  
  
3.  Cuando se te pida instalar el certificado caroot.cer, haz clic en **abrir**.  
  
4.  En la **certificado** cuadro de diálogo, haz clic en **instalar certificado**.  
  
5.  En el Asistente para importación de certificados, selecciona **máquina Local** como ubicación del almacén.  
  
6.  Selecciona **colocar todos los certificados en el siguiente almacén**y, a continuación, busca el **entidades de certificación raíz de confianza** ubicación.  
  
7.  Sigue las instrucciones para completar al asistente.  
  
    > [!NOTE]
    >  Una vez completada la instalación del certificado, iniciar sesión a la aplicación de mi servidor para Windows 8 o Windows 8.1 mediante el nombre del servidor y las credenciales de red.
